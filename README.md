# Распределённые системы — лабораторные работы ITMO

Репозиторий содержит реализации фундаментальных алгоритмов распределённых систем на Go с использованием фреймворка Hive.

Каждая директория — это отдельная лабораторная работа, реализующая конкретный алгоритм или протокол.

---

## Структура репозитория

### [deadlocked-mutex-ShaDi777](./deadlocked-mutex-ShaDi777/)
**Взаимное исключение и обнаружение дедлоков**

- Распределённый мьютекс по алгоритму **Maekawa**
- Распределённый снимок состояния по алгоритму **Chandy–Lamport**
- Обнаружение взаимоблокировок на основе глобального состояния (wait-for граф)

### [distributed-broadcast-ShaDi777](./distributed-broadcast-ShaDi777/)
**Надёжное причинно-следственное широковещание**

- **Reliable Broadcast** — надёжная рассылка с доставкой ровно один раз (crash-recovery модель)
- **Causal Broadcast** — причинный порядок доставки на векторных часах
- **Orderer** — линейное расширение частичного порядка и выявление параллельных сообщений

### [reliable-ordered-events-ShaDi777](./reliable-ordered-events-ShaDi777/)
**Надёжная доставка и причинный порядок**

- Надёжная доставка поверх fair-loss сети (ACK, дедупликация, повторные отправки)
- Векторные часы и отношение happens-before
- Выявление параллельных (конкурентных) событий
